You are an expert in OSHA enforcement and workplace safety.

Classify the citation narrative and extract structured hazard information.

CLASSIFICATION

Classify the citation as one of:
- FEDERAL_GDC
- STATE_GDC
- OSHA_STANDARD
- UNCERTAIN

Definitions:
- FEDERAL_GDC = Section 5(a)(1), general duty clause, or 29 USC 654(a)(1)
- STATE_GDC = state-plan general duty clause analogue
- OSHA_STANDARD = a specific OSHA or state safety standard is cited
- UNCERTAIN = cannot reasonably classify

Classification rules:
- If a specific OSHA or state standard is cited, classify as OSHA_STANDARD.
- If both general duty language and a specific standard appear, prefer OSHA_STANDARD.
- Set is_gdc = true only for FEDERAL_GDC or STATE_GDC.
- For OSHA_STANDARD, provide the cited or most likely standard in standard_guess.
- If no standard can be identified, use null.
- Only set state_reference = true if the narrative clearly indicates a state-plan citation or state analogue.
- Only populate state_name if explicitly stated or unmistakably implied.

HAZARD EXTRACTION

Extract the underlying hazard and separate it from the failure to control it.

hazard_text:
- One concise sentence, maximum 40 words.
- Describe the hazardous condition or exposure only.
- Focus on the physical, chemical, environmental, or behavioral hazard.
- Do not repeat statutory language.
- Do not describe the employer’s failure to train, implement a program, inspect, or provide PPE unless needed to understand the hazard.
- Do not include abatement instructions.

hazard_short:
- 2 to 6 words
- Hazard only, not the failure to control it

HAZARD CATEGORY

hazard_category must describe the hazard itself, not the failure.

Valid categories:
- fall
- struck_by
- caught_in_between
- electrical
- chemical_exposure
- fire_explosion
- oxygen_deficiency
- mobile_equipment
- machine_guarding
- confined_space
- ergonomic
- radiation
- thermal_burn
- heat_stress
- workplace_violence
- administrative
- other

Rules:
- Do not use training, program, procedure, or PPE as hazard_category.
- Example:
  "lack of fall protection training"
  => hazard_category = fall
- workplace_violence includes violent acts, threats, assaults, and domestic violence in the workplace.
- If the citation is administrative only and does not describe a physical hazard, use hazard_category = administrative.

FAILURE MODE

failure_mode describes why the hazard was not controlled.

Valid categories:
- training
- program
- procedure
- engineering_control
- PPE
- inspection_maintenance
- warning_alarm
- housekeeping
- unspecified
- other

Rules:
- Example:
  "lack of fall protection training"
  => failure_mode = training
- If the citation is administrative only, use failure_mode = program.

HAZARD SOURCE EXTRACTION

hazard_source_text:
- Extract any text identifying how the hazard was recognized.

hazard_source_list:
- Array of concise labels for recognized hazard sources.

Include ONLY if explicitly mentioned or clearly referenced:
- OSHA
- ANSI
- NFPA
- ASME
- ACGIH
- manufacturer instructions
- company policy
- industry practice
- prior incidents
- injury/illness records

Rules:
- Do not infer hazard sources from general safety language alone.
- If none are mentioned:
  - hazard_source_text = null
  - hazard_source_list = []

REASON SHORT

reason_short:
- Maximum 20 words
- Briefly explain the classification decision
- Do not restate the entire citation

OUTPUT REQUIREMENTS

Return ONLY valid JSON.
Do not include any text before or after the JSON.
Your entire response must be a single JSON object.
If a field is unknown, use null.
If no hazard source is present, use [] for hazard_source_list.
